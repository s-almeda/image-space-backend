üì° API Endpoints

GET /get-user/{id} - Get user data
curl -X GET https://snailbunny.site/artographer-data/get-user/shm
POST /add-user - Create new user
curl -X POST https://snailbunny.site/artographer-data/add-user \ -H "Content-Type: application/json" \ -d '{"userId": "newuser"}'
POST /add-user-image - Add user image
curl -X POST https://snailbunny.site/artographer-data/add-user-image \ -H "Content-Type: application/json" \ -d '{"userId": "shm", "image": {"userimage_id": "img_001", "title": "My Image", "imageUrl": "https://example.com/image.jpg", "worldCoords": [0.5, 0.3], "regionId": "1", "confidence": 0.95, "anchors": []}}'
POST /add-user-pin - Add pinned artwork
curl -X POST https://snailbunny.site/artographer-data/add-user-pin \ -H "Content-Type: application/json" \ -d '{"userId": "shm", "artwork": {"entryId": "art_001", "title": "Artwork", "image_urls": {}, "descriptions": {}, "artist": "Artist Name", "artist_names": [], "thumbnail_url": "", "url": "", "rights": "", "keywords": []}}'
POST /log-event - Log user event (accepts any arbitrary fields)
curl -X POST https://snailbunny.site/artographer-data/log-event \ -H "Content-Type: application/json" \ -d '{ "userId": "testuser", "system": "A", "taskNumber": 4, "message": "artwork clicked", "artworkId": "w_1235", "coords": "[0.23, 0.34]" }'
Note: System should be "A" (Artographer) or "B" (Baseline). Task number ranges from 1 (first task this participant completed that session) to 4 (4th task). You can include any number of additional arbitrary fields in the request.
GET /get-logs/{userId} - Get user logs
curl -X GET https://snailbunny.site/artographer-data/get-logs/shm
DELETE /remove-user-pin/{userId}/{entryId} - Remove pinned artwork
curl -X DELETE https://snailbunny.site/artographer-data/remove-user-pin/shm/art_001
DELETE /user/{id} - Delete user and all data
curl -X DELETE https://snailbunny.site/artographer-data/user/shm
